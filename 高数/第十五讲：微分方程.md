# 第十五讲：微分方程

> [!important] 核心目标
> 掌握各种阶数、各种类型的微分方程的求法。微分方程是考研数学中的高频考点，通常以大题或小题形式出现，且常与物理应用、几何应用结合。

## 一、 基本概念

### 1. 定义与构成
> [!note] 基础定义
> **微分方程**：含有未知函数、未知函数的导数（或微分）以及自变量的方程。
> **常微分方程**：未知函数是一元函数的微分方程。
> **阶**：微分方程中所出现的未知函数的**最高阶导数**的阶数。

### 2. 微分方程的解
> [!warning] 解的分类
> - **通解**：含有任意常数，且任意常数的个数与微分方程的阶数相同的解。
> - **特解**：确定了通解中任意常数后的解。
> - **初始条件**：给定自变量在某点处的函数值及各阶导数值。
> - **积分曲线**：微分方程解的图形。

## 二、 一阶微分方程

### 1. 可分离变量型
> [!danger] 形式与解法
> 形式：$y' = f(x)g(y)$
> 解法：
> 1. 分离变量：$\frac{dy}{g(y)} = f(x)dx$
> 2. 两边积分：$\int \frac{dy}{g(y)} = \int f(x)dx$
> 3. 整理得：$G(y) = F(x) + C$

### 2. 齐次型微分方程
> [!danger] 形式与解法
> 形式：$\frac{dy}{dx} = \varphi\left(\frac{y}{x}\right)$
> 解法：
> 1. 换元：令 $u = \frac{y}{x}$，则 $y = ux, \frac{dy}{dx} = u + x\frac{du}{dx}$
> 2. 代入：$u + x\frac{du}{dx} = \varphi(u) \Rightarrow x\frac{du}{dx} = \varphi(u) - u$
> 3. 分离变量：$\frac{du}{\varphi(u) - u} = \frac{dx}{x}$
> 4. 积分后回代 $u = \frac{y}{x}$。

### 3. 可化为齐次型的方程
> [!info] 扩展形式
> 形式：$\frac{dy}{dx} = \frac{a_1x + b_1y + c_1}{a_2x + b_2y + c_2}$
> 1. 若 $c_1 = c_2 = 0$，则为齐次型。
> 2. 若不全为 0，通过平移坐标轴消去常数项（解线性方程组确定平移量）。


### 3. 一阶线性微分方程
> [!danger] 标准形式与通解公式
> 形式：$y' + P(x)y = Q(x)$
> 通解公式：
> $$ y = e^{-\int P(x)dx} \left( \int Q(x) e^{\int P(x)dx} dx + C \right) $$

> [!tip] 常数变易法
> 对于非齐次方程 $y' + P(x)y = Q(x)$，先求对应的齐次方程 $y' + P(x)y = 0$ 的通解 $y = Ce^{-\int P(x)dx}$，再令 $C = C(x)$ 代入原方程求解。

### 4. 伯努利方程
> [!info] 形式
> 形式：$y' + P(x)y = Q(x)y^n \quad (n \neq 0, 1)$
> 解法：两边同除以 $y^n$，令 $z = y^{1-n}$，化为关于 $z$ 的一阶线性微分方程。

## 三、 可降阶的高阶微分方程

| 类型 | 特点 | 降阶方法 |
| :--- | :--- | :--- |
| **$y^{(n)} = f(x)$** | 右端仅含 $x$ | 连续积分 $n$ 次 |
| **$y'' = f(x, y')$** | 缺 $y$ | 令 $y' = p, y'' = \frac{dp}{dx}$ |
| **$y'' = f(y, y')$** | 缺 $x$ | 令 $y' = p, y'' = p\frac{dp}{dy}$ |

---

## 四、 高阶线性微分方程

### 1. 解的结构 (以二阶为例)
> [!important] 叠加原理
> 1. **齐次方程通解**：$y = C_1y_1(x) + C_2y_2(x)$ ($y_1, y_2$ 线性无关)
> 2. **非齐次方程通解**：$y = \bar{y} + y^*$ (齐次通解 + 非齐次特解)
> 3. **特解组合**：若 $y_1^*$ 是 $y''+py'+qy=f_1(x)$ 的特解，$y_2^*$ 是 $y''+py'+qy=f_2(x)$ 的特解，则 $y_1^*+y_2^*$ 是 $y''+py'+qy=f_1(x)+f_2(x)$ 的特解。

### 2. 二阶常系数齐次线性微分方程
> [!danger] 特征方程法
> 对应特征方程：$\lambda^2 + p\lambda + q = 0$
> 
| 特征根情况 | 通解 $y$ |
| :--- | :--- |
| $\Delta > 0$：两不等实根 $\lambda_1, \lambda_2$ | $y = C_1e^{\lambda_1x} + C_2e^{\lambda_2x}$ |
| $\Delta = 0$：二重实根 $\lambda_1 = \lambda_2 = \lambda$ | $y = (C_1 + C_2x)e^{\lambda x}$ |
| $\Delta < 0$：共轭复根 $\alpha \pm \beta i$ | $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$ |

### 3. 二阶常系数非齐次线性微分方程特解形式
> [!tip] 待定系数法
> - **$f(x) = P_n(x)e^{\alpha x}$**：
>   设 $y^* = x^k Q_n(x) e^{\alpha x}$，其中 $k$ 为 $\alpha$ 是特征方程根的重数（0, 1 或 2）。
> - **$f(x) = e^{\alpha x}[P_m(x)\cos\beta x + P_n(x)\sin\beta x]$**：
>   设 $y^* = x^k e^{\alpha x} [Q_l^{(1)}(x)\cos\beta x + Q_l^{(2)}(x)\sin\beta x]$，其中 $l = \max(m, n)$，$k$ 为 $\alpha \pm \beta i$ 是特征方程根的重数（0 或 1）。

## 五、 欧拉方程
> [!info] 形式
> $x^2y'' + pxy' + qy = f(x)$
> 解法：引入换元 $x = e^t$ (或 $t = \ln x$)，化为常系数线性微分方程。
