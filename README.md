# 考研数学复习笔记

## 项目简介

这是一个个人考研数学复习笔记项目，用于系统化地总结和复习考研数学知识点。项目采用Markdown格式编写，配合Obsidian进行知识管理和复习。

## 项目结构

```
note/
├── 高数/              # 个人总结的高数知识点
│   ├── 函数的极限.md
│   ├── 洛必达法则.md
│   └── 第六讲：中值定理、微分等式与微分不等式.md
├── 高数资料/          # 收集的参考资料（一般不主动查阅）
│   ├── advanced-math/
│   │   ├── exercise/
│   │   └── knowledge/
│   └── linear-algebra/
│       ├── exercise/
│       └── knowledge/
├── .obsidian/         # Obsidian配置
└── README.md          # 项目说明
```

## 目录说明

- **高数/**：核心内容目录，包含我自己的知识点总结和笔记。这是复习的主要材料。
- **高数资料/**：从别处收集的参考资料，包含高数和线性代数的知识点和习题。一般不主动查阅此目录，除非有特定需求。

## 笔记特点

- 使用Markdown格式，支持LaTeX数学公式
- 采用分讲式组织（如"第1讲"、"第六讲"）
- 包含tags和aliases用于Obsidian的双向链接
- 结构化知识点：基础定义、性质、计算方法、关联知识点

## 工具

- **笔记工具**：Obsidian
- **版本控制**：Git
- **格式**：Markdown + LaTeX

## 使用建议

- 主要关注 `高数/` 目录下的个人总结笔记
- `高数资料/` 目录作为参考，在需要时查阅
- 使用Obsidian的标签和链接功能进行知识关联
